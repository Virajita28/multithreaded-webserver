Our project focuses on developing a Multithreaded Web Server in C++ capable of handling multiple client requests simultaneously. It supports uploading and downloading of various file types, including videos and audio, while providing efficient concurrency through multithreading. The server implements HTTP protocol handling, error management, and logs client activity for monitoring. A simple frontend interface enables users to interact with the server and view real-time statistics. The system aims to be robust, scalable, and suitable for multimedia content delivery in local networks.
